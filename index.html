1 什么是 Github

Github 用于存放使用Git版本控制的软件代码和内容项目。于2008年2月运行, 根据维基百科, 2013年4月，GitHub用户数达到350万，代码库数量达到6百万个。

1.1 用来干嘛?

  • 个人和组织建立和存取代码库
  • 跟踪其他用户、组织、软件库的动态
  • 对软件代码的改动和 bug 提出评论
  • 显示开发者们怎样在代码库上工作以及软件的开发活跃程度

1.2 谁在使用?

  • Ruby , Ruby on Rails
  • PHP
  • JQuery
  • YUI
  • Bootstrap
  • hibernate

2.4 初始化版本库

如果是从头创建版本库，可以采用先克隆，建立提交数据，最后再通过推送完成GitHub版本库的初始化。步骤如下:

  • 克隆版本库 git@github.com:ndishare/Intro-to-github.git
  • 添加文件并提交 git add git commit -m "xxx"
  • 推送 git push origin master

2.3 建立主页

在GitHub用户通过创建特殊名称的Git版本库或在Git库中建立特别的分支实现对主页的维护

  • 创建个人主页
  • 创建项目主页
  • 使用 Jekyll 或者 hexo 维护

2.3.1 创建个人主页

  • 创建一个名为 ndishare.github.io 的Git版本库
  • 克隆新建立的版本库
  • 创建文件 index.html
  • 提交推送

2.3.2 创建项目主页

创建一个名为 gh-pages 的分支, 添加静态网页.

  • 创建分支 git symbolic-ref HEAD refs/heads/gh-pages
  • 删除缓存区文件 rm .git/index
  • 创建项目主页
  • 提交推送
